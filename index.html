FAQ

What is a rule-based AI agent and how is it used?
A rule-based AI agent is a software program that uses rules associated with the current economic outlook, technical data, user risk profile and other factors to assess the risk/reward outlook of a user's portfolio. It then recommends changes to the portfolio to enhance performance.
How are the recommendations validated?
The software has a module that constantly evaluates the actual performance against the predicted performance. Deviations result in adjustments to the parameters of the learning module of the agent agent. As the agent learns its performance is designed to improve.
